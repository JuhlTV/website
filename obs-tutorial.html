🚔 SHERIFF CHAT OVERLAY

Komplettes OBS-Setup Tutorial für Streamer

📋 Was ist das?

Das Sheriff Chat Overlay System ermöglicht es deinen Zuschauern, über die Website live Nachrichten zu schreiben, die sofort in deinem Twitch-Stream angezeigt werden. Perfekt für Interaktion mit deiner Community!

⚡ Echtzeit Nachrichten erscheinen sofort
🎨 Schön Authentisches Sheriff-Design
📱 Responsive Funktioniert auf allen Geräten
🔧 Einfach 3 Schritte zum Setup

🎯 Schritt 1: Zuschauer zur Chat-Seite einführen

Deine Zuschauer müssen die Chat-Seite besuchen, um Nachrichten zu schreiben:

YOUR_DOMAIN.com/chat

Beispiel: https://website-production-47e6.up.railway.app/chat

Du kannst diesen Link:

  • In der Twitch Panels setzen
  • In deiner Beschreibung verlinken
  • Im Stream-Title erwähnen
  • Als Command im Bot setzen (z.B. !chat)

🎬 Schritt 2: Overlay in OBS hinzufügen

Für OBS Studio (Windows/Mac/Linux):

  1. OBS öffnen und deine Scene auswählen
  2. Quelle hinzufügen: Im "Quellen"-Panel auf + klicken
  3. "Browser" auswählen (wenn nicht sichtbar, "Weitere Quellen" auswählen)
  4. Quelle benennen: z.B. "Sheriff Chat Overlay"
  5. URL eingeben:
    https://website-production-47e6.up.railway.app/overlay
    (Oder für lokales Testing: http://localhost:3000/overlay)
  6. Breite: 1920 px
  7. Höhe: 400 px
  8. FPS: 30
  9. OK klicken

Quelle positionieren:

  1. Das Overlay sollte jetzt in deiner Scene sichtbar sein
  2. Mit der Maus verschieben (ziehen)
  3. Mit den Ecken die Größe anpassen
  4. Optimal unten auf dem Stream platzieren (wie ein Twitch-Chat)
✅ Fertig! Dein Overlay ist jetzt in OBS eingebunden und zeigt Live-Chat-Nachrichten

🔧 Schritt 3: Einstellungen anpassen

Optional: Overlay-Größe anpassen

Je nach deiner Stream-Auflösung und Design kannst du die Größe ändern:

  • 1920x1080 Stream: 1920x400 empfohlen (ganzeBreite unten)
  • 1280x720 Stream: 1280x300 empfohlen
  • Oben rechts Platzierung: 500x350 empfohlen
  • Kleine Corner: 400x250 empfohlen

Refresh-Rate

Die FPS-Einstellung in OBS:

  • 30 FPS: Standard (weniger CPU-Last)
  • 60 FPS: Für flüssigere Animationen (höhere CPU-Last)
⚠️ Wichtig: Der Hintergrund ist transparent! Das heißt, das Overlay passt sich automatisch an deinen Stream-Hintergrund an

💡 Pro-Tipps

1. Chat-Link prominent anzeigen

  • Erstelle einen Twitch Panel mit dem Chat-Link
  • Oder setze ihn als Custom Command im Streamlabs Bot
  • Beispiel: !chat Schreib deine Nachricht im Chat: [URL]

2. Mehrere Overlays

Du kannst das Chat-Overlay mehrfach hinzufügen:

  • Unten für den Hauptchat
  • Oben rechts für wichtige Nachrichten
  • Seitlich für "Alerts"

3. Responsive Design

Das Overlay passt sich automatisch an unterschiedliche Fenstergrößen an - perfekt für verschiedene Stream-Setups!

4. Browser-Quelle neuladen

Falls das Overlay nicht mehr aktuell ist:

  1. Rechtsklick auf die Browser-Quelle in OBS
  2. "Interact" auswählen
  3. F5 zum Neuladen drücken

🆘 Troubleshooting

❌ Overlay wird nicht angezeigt?
→ Überprüfe, dass die URL korrekt ist (https:// vs http://)
→ Prüfe, ob OBS Internetzugriff hat (Firewall-Einstellungen)
→ OBS neu starten
❌ Nachrichten aktualisieren nicht?
→ Rechtsklick auf Browser-Quelle → "Refresh Cache"
→ Überprüfe deine Internetverbindung
→ Localhost statt domain verwenden zum Testen
❌ Overlay ist zu klein/groß?
→ Ändere die Breite/Höhe in den OBS-Quellen-Einstellungen
→ Verwende die Ecken zum Skalieren
❌ WebSocket-Fehler in OBS-Konsole?
→ Der Server läuft möglicherweise nicht
→ Überprüfe, ob Railway deployt hat
→ Warte 1-2 Minuten nach dem Deployment
❌ Zu viele Nachrichten?
→ Das ist kein Bug - das ist Feature! 🎉
→ Das System zeigt max. 8 Nachrichten gleichzeitig
→ Ältere Nachrichten werden automatisch entfernt
