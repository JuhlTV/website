# Sheriff Manfred Mainke - Official Website

Eine professionelle Website im Stil eines Texas Sheriff Departments für Sheriff Manfred Mainke.

## 🌟 Features

- **Authentisches Texas Sheriff Department Design** mit klassischen Farben (Navy Blau, Gold)
- **Texas Star Badge** als zentrales Element mit Animation
- **Responsive Design** - funktioniert auf allen Geräten
- **Interaktive Elemente**:
  - Smooth Scrolling Navigation
  - Animierte Statistiken
  - Rotierende Sicherheitshinweise
  - Kontaktformular mit Benachrichtigungen
  - Parallax-Effekt im Hero-Bereich
  - Fade-in Animationen beim Scrollen

## 📁 Dateien

- `index.html` - Hauptseite mit allen Sektionen
- `styles.css` - Komplettes Styling im Sheriff Department Design
- `script.js` - Interaktive Funktionen und Animationen

## 🚀 Verwendung

Öffnen Sie einfach die `index.html` Datei in einem modernen Webbrowser.

## 📱 Sektionen

1. **Header** - Mit Texas Star Badge und Notfallnummer
2. **Navigation** - Sticky Navigation mit allen wichtigen Links
3. **Hero** - Eindrucksvoller Willkommensbereich
4. **Sheriff** - Profil von Sheriff Manfred Mainke mit Statistiken
5. **Department** - Übersicht der Abteilungen
6. **Services** - Angebotene Dienstleistungen
7. **Contact** - Kontaktinformationen und Formular
8. **Footer** - Zusätzliche Links und Informationen

## 🎨 Design-Elemente

- **Farben**: Navy Blau (#1a3a52), Gold (#c9a961), Rot (#8b0000)
- **Schriftarten**: 'Crimson Text' für Überschriften, 'Open Sans' für Text
- **Icons**: Emoji-basierte Icons für moderne Darstellung
- **Animationen**: Subtile, professionelle Animationen

## 🌐 Browser-Kompatibilität

Funktioniert in allen modernen Browsern:
- Chrome
- Firefox
- Safari
- Edge
